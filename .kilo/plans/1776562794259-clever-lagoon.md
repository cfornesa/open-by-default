# Fix Hamburger Menu Visibility Plan

## Problem

The hamburger menu icon is not visible in mobile view because:
1. The `.menu-toggle` class has `display: none` by default
2. The mobile breakpoint (@media max-width: 767px) only updates styling but doesn't change the display property to make it visible

## Solution Approach

### 1. Mobile-First Display Strategy
- Change default `.menu-toggle` to `display: block` for mobile-first approach
- Use media queries to hide it on desktop where navigation links are visible

### 2. Enhanced Mobile Visibility
- Add proper display property for mobile view
- Ensure the hamburger menu is positioned correctly within the navigation container
- Maintain all existing styling enhancements

### 3. Desktop Navigation Logic
- Hide hamburger menu on desktop viewports
- Ensure navigation links are visible on desktop
- Maintain proper layout and spacing

## Implementation Details

### CSS Changes Required

#### 1. Update Default Menu Toggle Display
```css
/* Menu Toggle Button - Mobile First */
.menu-toggle {
  display: block;  /* Changed from none to block */
  background: linear-gradient(135deg, var(--color-primary-1), var(--color-primary-2));
  border: 2px solid var(--color-text);
  border-radius: 8px;
  padding: 8px;
  margin: -8px;
  cursor: pointer;
  z-index: 100;
  position: relative;
  transition: all var(--transition-base);
}
```

#### 2. Hide on Desktop Viewports
```css
/* Hide hamburger menu on desktop */
@media (min-width: 768px) {
  .menu-toggle {
    display: none;
  }
}
```

#### 3. Update Mobile Container Layout
```css
/* Mobile Header Layout - Enhanced */
@media (max-width: 767px) {
  .nav-container {
    background: linear-gradient(135deg, var(--color-primary-1), var(--color-primary-2));
    border: 2px solid var(--color-text);
    padding: var(--space-md);
    justify-content: space-between;
    align-items: center;
  }
  
  .site-title {
    margin-left: var(--space-lg);
    font-size: 1.25rem;
    font-weight: 700;
    color: var(--color-text);
    text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.3);
  }
  
  .menu-toggle {
    margin-right: var(--space-lg);
    background: var(--color-text);
    border: 2px solid var(--color-primary-1);
  }
  
  /* Navigation menu for mobile */
  .nav-links {
    position: absolute;
    top: 100%;
    left: 0;
    right: 0;
    flex-direction: column;
    background-color: var(--color-bg);
    border-bottom: 1px solid var(--color-border);
    padding: var(--space-xl) var(--space-lg);
    gap: var(--space-lg);
    transform: translateY(-100%);
    opacity: 0;
    visibility: hidden;
    transition: transform var(--transition-base), opacity var(--transition-base), visibility var(--transition-base);
    z-index: 50;
  }
}
```
