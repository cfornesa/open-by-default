# Enhanced Design Plan - Tri-Color Scheme & Mobile Header Layout

## Overview
Implement a vibrant tri-color scheme using primary colors as base with black, gray, and white accents, while fixing the mobile header layout to position the logo and hamburger menu on opposite horizontal ends with proper padding.

## Current State Analysis

### Design Assessment
- **Current Palette**: Dark theme (#111013 bg, #e2e0e7 text, #d49a6a primary) - muted and safe
- **Mobile Header**: Logo and hamburger menu too close together, both centered
- **Layout Philosophy**: "Open canvas" with asymmetric, expressive compositions

### User Requirements
- More contrast and color variance compared to current design
- Tri-color scheme with primary colors as main colors
- Accent with black, gray, and/or white
- Mobile header: logo left edge, hamburger menu right edge, both with padding

## Design Decisions (To Be Confirmed)

### Color Strategy
- **Primary Base Colors**: Three vibrant primary colors for maximum contrast and impact
- **Accent Colors**: Black (#000000), various grays (#333333, #666666, #999999), and white (#ffffff)
- **Accessibility**: WCAG AA compliance for all text contrast ratios
- **Bombastic Direction**: Enhanced visual drama while maintaining readability

### Mobile Header Layout
- **Breakpoint**: Mobile (< 768px)
- **Logo Position**: Far left edge with var(--space-md) padding
- **Hamburger Menu**: Far right edge with var(--space-md) padding
- **Navigation Container**: Justify-content: space-between for proper spacing
- **Responsive Behavior**: Maintains layout across all mobile viewports

## Technical Implementation Strategy

### CSS Architecture
- **Variable System**: Enhanced CSS custom properties for color management
- **Mobile-First**: Responsive design starting from mobile breakpoint
- **Component-Based**: Modular CSS for easy maintenance
- **Accessibility-First**: WCAG AA compliance baked into design decisions

### Color System Structure
```css
:root {
  /* Tri-Color Primary Scheme */
  --color-primary-1: #ff4d4d;    /* Vibrant Red */
  --color-primary-2: #4d4dff;    /* Vibrant Blue */
  --color-primary-3: #44ff44;    /* Vibrant Green */
  
  /* Accent Colors */
  --color-accent-black: #000000;
  --color-accent-gray-1: #333333;
  --color-accent-gray-2: #666666;
  --color-accent-gray-3: #999999;
  --color-accent-white: #ffffff;
  
  /* Enhanced Semantic Colors */
  --color-border-primary: var(--color-primary-1);
  --color-accent-hover: var(--color-primary-2);
  --color-text-muted: var(--color-accent-gray-2);
}
```

### Mobile Header Layout
```css
@media (max-width: 767px) {
  .nav-container {
    justify-content: space-between;
  }
  
  .site-title {
    margin-left: var(--space-md);
  }
  
  .menu-toggle {
    margin-right: var(--space-md);
  }
}
```
