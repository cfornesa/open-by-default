What Tools Are Available?

The following are open-source AI tools that I have tested and recommend.

Free Open-Source AI Tools

💻

Opencode Zen

Opencode Zen is another provider that offers access to free and open-source AI models with higher rate limits compared to OpenRouter (due to model-specific rate limits, rather than overall rate limits).

The main caveat, however, is that its free models are time-limited and are intentionally provided for free in exchange for their partnering AI firms' data collection schemes.

Otherwise, if you are building a small, personal project, but you don't have an unlimited budget, Opencode Zen might be a good option to consider.

🔧

Deepseek

Deepseek offers a world-renown chatbot interface and mixture-of-experts models that disrupted OpenAI's dominance in the AI industry. The best part? It's completely free to use and extremely cheap to implement in your RAG-based chatbots or other applications.

🔧

Le Chat

Le Chat is a user-friendly interface for interacting with Mistral AI's open-source models, making it easy to experiment with different options without incurring costs.

🛠️

OpenRouter

OpenRouter provides access to a wide range of open-source AI models. With a one-time $10 credit purchase, you can get upgraded access to these models.

🔧

Kilo Code

Kilo Code, itself, is a powerful VS Code extension for working with AI models with a user interface that's friendly and intuitive.

🚀

Kilo Gateway

Kilo Gateway offers free access to numerous AI models, making it easy to experiment with different options without incurring costs.

Paid Open-Source AI Tools

🔧

Le Chat Pro

Mistral AI provides both proprietary and open-source models.

Excellent open-source options include Mistral AI's open-source mixture-of-experts models, while Vibe CLI runs on Mistral AI's open-source Devstral model and provides an excellent alternative to Big Tech's agentic coding offerings.

Access to Mistral AI's premium chat interface and Vibe CLI program are included with Le Chat Pro.

🛠️

Opencode Go

Opencode Go offers a comprehensive suite of tried-and-true, open-source AI models meant for agentic coding, with model access from providers ranging from Qwen to OpenAI.

🛠️

Gemini CLI

Gemini CLI provides a command-line interface for interacting with Google's Gemini AI models, allowing users to leverage these powerful tools directly from their terminal. While there is a generous free-tier, upgrading provides access to more advanced features and higher usage limits.
