What Tools Are Available?

Some basic information about what each type of open-source AI tool can do for you.

Free Tools

🔧

Deepseek

Deepseek offers a world-renown chatbot interface and mixture-of-experts models that disrupted OpenAI's dominance in the AI industry. The best part? It's completely free to use and extremely cheap to implement in your RAG-based chatbots or other applications.

🔧

Kilo Code

Kilo Code, itself, is a powerful VS Code extension for working with AI models with a user interface that's friendly and intuitive.

🛠️

OpenRouter

OpenRouter provides access to a wide range of open-source AI models, making it easy to experiment with different options without incurring costs.

🚀

Kilo Gateway

Kilo Gateway offers free access to numerous AI models, making it easy to experiment with different options without incurring costs.

💻

Opencode Zen

Opencode Zen is another provider that offers access to free and open-source AI models with higher rate limits compared to OpenRouter (due to model-specific rate limits, rather than overall rate limits).

Paid Tools

🔧

Mistral AI

Mistral AI provides both proprietary and open-source models. Excellent open-source options include Mistral AI's open-source mixture-of-experts models, while Vibe CLI runs on Mistral AI's open-source Devstral model and provides an excellent alternative to Big Tech's agentic coding offerings.

🛠️

Opencode Go

Opencode Go offers a comprehensive suite of tried-and-true, open-source AI models meant for agentic coding, with model access from providers ranging from Qwen to OpenAI.

🚀

Kilo Gateway

Kilo Gateway offers free access to numerous AI models, making it easy to experiment with different options without incurring costs.
